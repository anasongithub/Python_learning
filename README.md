# Python Learning & Practice Repo

Welcome to your Python learning repository! This project contains practice scripts and exercises organized by topic.

---

## Topics Covered & Roadmap

Below is a summary of the topics you've covered so far, followed by empty rows for future topics you plan to study.

| Topic | Key Concepts Covered |
| :--- | :--- |
| **Variables** | Variable declarations, basic data types, type casting, and user inputs. |
| **Lists** | List creation, mutability, appending/inserting, and popping/removing elements. |
| **Tuples** | Tuple definition, immutability, methods (`count`, `index`), and helper functions (`len`, `max`, `min`). |
| **Conditionals** | Decision making (`if`, `elif`, `else`), nested conditions, and membership checks. |
| **Dictionaries** | Key-value pairs, modifying/querying data, and built-in dictionary methods. |
| **Sets** | Set creation, uniqueness constraint, and set properties. |
| **Functions** | Function declarations, parameters, list/string filtering, and recursion. |
| **Loops** | Loops (`for` and `while`), range(), break statements, and console patterns. |

## How to Run the Practice Code

To run any of the scripts, navigate to the repository root directory in your terminal and run:

```bash
python3 <directory_name>/<file_name>.py
```
